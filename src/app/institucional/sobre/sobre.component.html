Instruções para participação

Os atletas que decidem participar das atividades organizada pelos Brutos do Mato, devem possuir os seguintes equipamentos: Mochila de hidratação - preferencialmente coletes - de no mínimo 1 litro para percursos médios e 1,5 litros para percursos longos. Tênis de trail run com gripes adequados para terrenos montanhosos, rochosos, arenosos e preparados para molhar. Além disto, devem se munir de energéticos para reposição de eletrólidos, barras de cereal, cápsulas de sais minerais, oleaginosas, frutas, dentre outros itens. Os mais experientes estarão sempre acessíveis para indicar os equipamentos e suprimentos necessários a cada peleja.
